# 🕌 Mezquita Abdullah - Sitio Web

Sitio web moderno y multilingüe para la Mezquita Abdullah, diseñado para acoger a visitantes no musulmanes y presentar el Islam de manera amigable, honesta y atractiva.

## 📋 Características

- ✅ **Diseño moderno y responsive** (móvil, tablet, escritorio)
- ✅ **4 idiomas**: Español, Inglés, Árabe (RTL), Francés
- ✅ **Cambio de idioma dinámico** sin recargar la página
- ✅ **Secciones interactivas**:
  - Hero con bienvenida multilingüe rotatoria
  - Contador animado de estadísticas
  - Tarjetas de preguntas con efecto flip
  - Galería de imágenes de la mezquita
  - Formularios de visita y consulta al imam
- ✅ **Estética islámica** con paleta verde, dorada y patrones geométricos
- ✅ **Optimizado para SEO** y accesibilidad

## 🚀 Cómo usar

1. Descomprime el archivo ZIP
2. Abre `index.html` en cualquier navegador moderno
3. ¡Listo! No requiere servidor ni configuración

## 🎨 Paleta de colores

- **Verde primario**: `#0F5C4E` (Islam, paz)
- **Dorado**: `#D4AF37` (elegancia)
- **Azul oscuro**: `#1a2a4a` (footer)
- **Crema**: `#FAF7F2` (fondos suaves)

## 📝 Personalización

### Cambiar textos
Edita las traducciones en `js/translations.js` para cada idioma.

### Cambiar imágenes
Reemplaza los archivos en la carpeta `images/` manteniendo los mismos nombres.

### Información de contacto
Edita la sección `<footer>` en `index.html` para añadir teléfono, dirección y email reales.
